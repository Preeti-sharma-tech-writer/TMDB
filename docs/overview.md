# Overview

## About TMDB
The Movie Database (TMDB) is an entertainment explorer platform containing details about movies, tv shows, awards, people and many more. The collection of metadata also includes high resolution posters and backdrops. Released in 2008, it is currently being consumed by over 1,500,000 developers and companies. The metadata is available in over 39 international and regional languages making it accessible across 180 countries. 

### API Documentation
Using these APIs you can [search](./reference/search-movies.md) for a specific movie, [list](./reference/popular-movies.md) currently popular movies, retrieve all [metadata](./reference/movie-details.md) for a specific movie, look up the [list of genres](./reference/genre-list.md) and their codes, and [discover](./reference/discover-movies.md) movies based on numerous filters and sorting options. In order to integrate the APIs efficiently, refer the [Configuration API](./reference/configuration.md). It provides the additional information needed to build the complete APIs. 
To begin, refer to the Getting Started guide first. It explains how to authenticate your requests.
If you get stuck or need assistence, the [API Support](https://www.themoviedb.org/talk/category/5047958519c29526b50017d6) forum is very active and resolves queries promptly.
